# switor

Switor app is a mac os that can change display resolution and refresh rate easily, especially using a shortcut key.

## Features

- [ ] List display resolution for monitors connected to the mac (resolution and refresh rate)
- [ ] Change display resolution for monitors connected to the mac with UI
- [ ] Change display resolution for monitors connected to the mac with shortcut key
- [ ] Can add the app to launch on the start up

## UI

- [ ] can change display resolution and refresh rate with slider and choose option from list
- [ ] can setup the shortcut key in the `.config` folder to change display resolution and refresh ratespecific for monitor
